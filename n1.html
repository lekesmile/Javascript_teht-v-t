Tehtävä 1

Sinun on luotava javascript-toiminto, joka laskee välillä 0 - 100. Kun saavutat 100, laskuri palaa arvoon 0 ja laskee jatkuvasti alusta. Näytät laskurin selaimen konsolissa nopeudella 120 ms.

You have to create a javascript function that counts from 0 to 100. When reaching 100, the counter gets back to 0 and keeps counting from start again. You will display the counter in the console of the browser with a speed of 120 ms.

Open the console to view the result !
