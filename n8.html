Tehtävä 8

Sinun on luotava toiminto, joka korvaa merkkijonon kaikki "e" n merkinnällä "o". Tämä toiminto suoritetaan, kun napsautetaan “Muunna” -painiketta ja otetaan argumentiksi kirjoitettu merkkijono (tekstinsyöttö). Muunnettu merkkijono on sitten näytettävä selaimen konsolissa.

You have to create a function which replaces all “e” in a string by “o”. This function is executed when clicking the “Transform” button and take the typed string (text input) as an argument. The modified string has then to be displayed in the console of the browser.
